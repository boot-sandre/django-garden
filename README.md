# Django api server + single page app template


## To get started:

        cd gutenberg
        yarn dev

## To build & start for production:

        cd gutenberg
        yarn build
        yarn start

## To test:

        cd gutenberg
        yarn test

- **Api server** using [Django Ninja](https://github.com/vitalik/django-ninja)
